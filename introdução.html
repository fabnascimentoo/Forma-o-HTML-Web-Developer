Falando sobre tags

Voltar menu
São responsáveis por estruturar o que cada coisa faz dentro do código, e como será interpretada pelo navegador. Existem inúmeras tags, cada uma com funções diferentes dentro do código.

Atributos das tags

Voltar menu
Existem algumas tags que necessitam de "atributos", estes funcionam como especificações de como irá ser feita a formatação de determinado texto ou imagem por exemplo, de como cada elemento se comportará dentro de uma página.

Listas ordenadas e não ordenadas

Voltar menu
Listas ordenadas: são como o próprio nome diz "organizadas" por número: 1º, 2º, 3º... Litas não ordenadas: não possuem números que ordenem seus itens mas sim marcadores (algumas bolinhas) que podem ser modificados posteriormente para outro tipo.
