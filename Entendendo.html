História da web

Voltar menu
Para a internet chegar onde estamos atualmente houveram muitos passos durante vários anos. Um grande exemplo disso é a evolução dos computadores que começou lá por volta de 1946 com o primeiro computador o ENIAC, depois dele a evolução só continuou, grandes empresas de tecnologias foram criadas e não somente o computador acessa a internet mas vários outros dispositivos também. O computador foi ficando cada vez menor e a tecnologia dentro dele cada vez maior.

O que são clients

Voltar menu
Clients, podemos dizer que o cliente é o receptor da programação mais visual. É nele onde o usuário irá visualizar o site, o aplicativo, etc. Existem linguagens de programação específicas para o cliente.

O que são servers

Voltar menu
Servers, são os computadores responsáveis por atender os pedidos requisitados pelo cliente. Podem fornecer recursos de banco de dados, redes, etc.

Linguagem de programação

Voltar menu
Podem ser de baixo ou alto nível, quanto mais alto o nível da linguagem de programação, maais próxima da linguagem humana ela é. Computadores interpretam linguagens de programação. Existem várias dessas linguagens sendo as mais populares: Java, C#, Php, entre outros.
